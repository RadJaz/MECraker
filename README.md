# MECraker

MECraker is a set of python tools for collecting campaign finance data from the Missouri Ethics Commision website.

## Dependencies

- Python 3
- [MuPDF](https://mupdf.com/releases/index.html)

### PIP module dependecies

- requests
- PyMuPDF

## Use

To use as a program:

```python3 -m mec```
